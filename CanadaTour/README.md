# Android - Canada Tour App

## Install

* Android Studio 2.3 or Higher
* Build Tools 23+
* Target SDK: API Level 26
* ``` git clone https://github.com/Alifyz/Android.git ```

## Libraries & Features

* No external Libraries were used in this App. 
* ListView - with Dummy Data
* This app uses ViewPager with Dynamic Swining Gesture to change the active Fragment
