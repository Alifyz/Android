# Android MVP 

This project is meant to be an example of the MVP pattern for Android projects. 

Este repositório é destinado ao padrão MVP em projetos Android. 

# Pre-requisites

  - Android Studio 2.3+
  - Android SDK Build Tools 23+
  - Min SDK Level: 19
